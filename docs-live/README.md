# mc-pulse docs

Documentation for the mc-pulse API, built with [Docusaurus](https://docusaurus.io/).

Run these commands from the `docs` directory.

## Install
```bash
npm install
```

## Local development
```bash
npm run start
```
Opens a local dev server with live reload.

## Build static site
```bash
npm run build
npm run serve # to preview the built output
```

The `package.json` enforces **Node.js 20+** for the docs site.
